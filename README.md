# phase-1-project

**_ Note _**
Due a to high volume of API data, the application may take a few seconds to load all contents

APPLICATION OVERVIEW

-   A character look-up application utilizing Marvel API.
-   Application displays every Marvel characters introduced in a grid card format.
-   Clicking each card will lead to their respective details page in the official Marvel website.
-   Searching a character in search box will display a result card with extra information of their background and comic book appearances if applicable.

There are three addEventListner methods used in this application

1. DOMContentLoaded
2. Submit event for a search button
3. Click event for X search result close button
